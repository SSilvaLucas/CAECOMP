Evento

A Semana de Integração de Tecnologia e Empreendedorismo objetiva se a proporcionar um ambiente de integração e crescimento mútuo entre os participantes congregando alunos, professores, engenheiros e demais profissionais que venham a se interessar, para implementar a integração social, tecnológica e profissional, o intercâmbio de experiências e reciclagem de conhecimentos.

Eventos acadêmicos voltados à integração são importantes vias de aprendizagem e conhecimento para acadêmicos, a premissa de membros de uma mesma comunidade possibilita a ampliação de comunicação pessoal, assim, permitindo a troca de informações de maneira intensa. Para a formação de uma atmosfera de desenvolvimentos universitários é imprescindível que os membros constituintes desse meio possuam um eficaz canal de controle, auxiliando o processo de desenvolvimento e maximizando os resultados.

Assim, a comissão organizadora da 3ª Semana de Integração de Tecnologia e Empreendedorismo, formada por membros do Centro Acadêmico de Estudantes de Engenharia de Computação promove, em conjunto com o Centro de Educação Empreendedora - CEE (UEPG), e com o apoio da Universidade Estadual da Ponta Grossa (UEPG), a 3ª Semana de Integração de Tecnologia e Empreendedorismo, na cidade de Ponta Grossa - PR, no período de 06 a 10 de novembro de 2017, promovendo a interação entre estudantes, pesquisadores e profissionais na área da engenharia.

Para esta edição, todas as atividades e minicursos serão GRATUÍTOS. Desta maneira, fica a responsabilidade ao participante em apenas se inscrever pela página web do evento para garantir vaga nos minicursos e certificado de participação (mediante a 70% de freqência).

Resumo

Big Data na Saúde
arrow_drop_down
Resumo

Existem três áreas auspiciosas para o uso de Big Data em saúde: medicina de precisão; prontuários eletrônicos do paciente; e internet das coisas. Em relação às novas técnicas estatísticas, espera-se que técnicas de machine learning (principalmente as árvores de classificação e regressão), metodologias para controlar por associações espúrias (como a correção de Bonferroni e a taxa de falsas descobertas) e metodologias para a redução da dimensão dos dados (como a análise de componentes principais e o propensity score matching) sejam cada vez mais utilizadas. Essas e outras aplicações serão apresentadas nesta palestra ministrada por Stephen.

Sobre o Palestrante

Stephen Kunihiro é graduado em Odontologia pela UNESP, Mestre pela UNICAMP e com duas Pós Graduação (USP, UNICAMP). É sócio proprietário das empresas NK Solutions e Eneka Odontologia. Envolvido com projetos de startups, vive em busca de inovação e tecnologia aplicada a saúde.

Sobre

Git é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase em velocidade. O Git foi inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento do kernel Linux, mas foi adotado por muitos outros projetos. Framework Laravel
arrow_drop_down
Sobre

Laravel é um Framework PHP utilizado para o desenvolvimento web, que utiliza a arquitetura MVC e tem como principal característica ajudar a desenvolver aplicações seguras e performáticas de forma rápida, com código limpo e simples, já que ele incentiva o uso de boas práticas de programação e utiliza o padrão PSR-2 como guia para estilo de escrita do código. Para a criação de interface gráfica, o Laravel utiliza uma Engine de template chamada Blade, que traz uma gama de ferramentas que ajudam a criar interfaces bonitas e funcionais de forma rápida e evitar a duplicação de código. Para se comunicar com um Banco de Dados o Laravel utiliza uma implementação simples do ActiveRecord chamada de Eloquent ORM, que é uma ferramenta que traz várias funcionalidades para facilitar a inserção, atualização, busca e exclusão de registros. Com configuração simples e pequena e com pouco código podemos configurar a conexão com Banco de Dados e trabalhar com ele.

Esse minicurso visa introduzir o conceito de Framework MVC, assim como mostrar o passo a passo básico de como criar um CRUD.

A quem se destina?

Minicurso voltado para alunos de terceiro e quarto ano que utilizarão o Framework Laravel na matéria de Projetos. Para participar, o aluno deverá dominar o básico de tecnologias web e php orientado a objeto.

Ementa
  • Introdução sobre Frameworks MVC.
  • Explicação do que é necessário num ambiente para desenvolver em Laravel.
  • Mostrar fluxo de dados dentro do Framework.
  • Como utilizar o Query Builder.
  • O que é Eloquent e como ele facilita a manipulação de dados.
  • O que são controladores.
  • Organizando Layouts para facilitar o desenvolvimento frontend.
  • Submetendo formulários na sua aplicação.
  • Validação de formulários.
  • Relacionamentos Eloquent.
  • Mostrar autenticação padrão disponibilizado pelo Laravel.
